## ⚙️ How to Use Orchid_Panda

Once running, Orchid_Panda shows a list of active programs and processes on your PC. It updates automatically.

- The main screen displays process names, CPU, and memory use.
- You can sort processes by clicking the column headers.
- If a program crashes or slows down, Orchid_Panda alerts you.
- Use the "Watchdog" feature to restart specific programs automatically if they stop working.
- The app logs process events so you can see what happened over time.
